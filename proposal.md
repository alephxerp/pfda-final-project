# Point and Click Prototype

## Repository
https://github.com/alephxerp/pfda-final-project

## Description
For this project, I plan on creating a simple, one-screen point-and-click game  using Pygame. This will include many core features of interactive media, but primarily focus on UI, which is a big part of digital arts.

## Features
- Game Menu
	- A menu screen that can be toggled by user input. Will have buttons for basic settings (fullscreen, game speed, etc.) and closing the program.
- Point-and-click Elements
	- Elements in the game world that can be clicked on with the mouse. These elements should bring up dialogue or execute some function when interacted with.
- Dialogue
	- A visual novel-style system that displays text in a dialogue box. Is called from point-and-click elements and shows different text based on what is clicked. 

## Challenges
- Implementing dynamic font and text in Pygame.
- Creating functional and intuitive UI.
- Short sentence of any skill or topic that needs to be learnt or researched.

## Outcomes
Ideal Outcome:
- A short point-and-click game, featuring dialogue and a menu. Elements should have hover and click effects, and be able to trigger dialogue and other logic.

Minimal Viable Outcome:
- Interactive elements that display text when clicked and a simple menu system.
